# Анализ маркетинга и прогнозирование

## Что содержат файлы

| Файл | Что это |
|------|---------|
| `Без названия (1).ipynb` | Jupyter Notebook с кодом на Python (расчёты, графики, модели) |
| `forecast_chan.xls` | Прогнозы по каналам продаж / рекламы |
| `market_file.xls` | Маркетинговые данные: расходы, лиды, охваты |
| `market_money.xls` | Финансы маркетинга: бюджет, ROI, CAC |
| `market_time.xls` | Даты и временные метки маркетинговых кампаний |
| `money.xls` | Основные финансовые показатели (выручка, прибыль) |

## Как запустить
1. Установите Python и Jupyter.
2. Поместите все файлы в одну папку.
3. Откройте `Без названия (1).ipynb` и выполните ячейки.
